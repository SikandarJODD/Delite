# Create Sveltekit + Tailwind CSS Application

## Installation

##### <code>folder-name</code> can be anything

```bash
  git clone https://github.com/SikandarJODD/SvelteKit-min.git folder-name
```

##### Install the code using

```bash
  npm i
```

##### Run the Code

```bash
  npm run dev -- --open

```
